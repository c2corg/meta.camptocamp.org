meta.camptocamp.org est un méta-moteur de partage de conditions de courses en montagne.
Les sites partenaires alimentent le méta-moteur avec leur sorties (un résumé seulement).
Ce formulaire permet ensuite d'obtenir un fil RSS contenant les dernières sorties correspondant à vos critères.
Le fil RSS peut être utilisé dans un agrégateurs RSS, ou pour alimenter un contenu dans une page web.

Avec certains navigateurs, cliquez dans les champs avec la touche CTRL enfoncée pour faire de multiples (dé)sélections.

Activités

Pour l'instant, les activités ne sont pas définies de la même façon pour tous les sites partenaires :
  • Pour montagneinfo.net, l'alpinisme rocheux (correspondant à "Escalade en montagne" sur camptocamp.org) est réparti entre "Alpinisme neige, glace, mixte" et "Escalade en montagne".

Pour qu'une date soit prise en compte, les 3 champs doivent être définie. Définir au choix :
- aucune date : la recherche se fera sur toutes les dates,
- la date de début : la recherche se fera à partir de cette date,
- la date de début et de fin : la recherche se fera entre ces 2 dates.
